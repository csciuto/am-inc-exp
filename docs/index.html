X axis
Household income on a log scale. Drag to select a range.
Y axis
Frequency — taller clusters mean more households near that income. Dots are also spread apart vertically to reduce overlap, so the exact y position of each dot isn't meaningful.
Dot size
Proportional to the household's sampling weight. A larger dot represents more US households.
Colors
Set by the Group tab. Click a legend swatch to filter by that category. Topcoded incomes (⊕) are censored at the survey year's reported maximum.
Dot count / Static / All
1K / 3K / 5K sets the maximum dots drawn. Static draws randomly from all households and dims non-matching ones as you adjust filters — the same dots persist as you click around within a state or national view. All draws only from households that match your current filters, up to the dot limit; useful when zoomed into a state or narrow filter where you want to see the actual population.
Data
IPUMS CPS ASEC survey years 2021–2025 (income years 2020–2024), pooled. All statistics are weighted. Small samples are flagged — see the reliability indicator.
